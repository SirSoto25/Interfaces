SASS SASS es una gran herramienta, escrita usando Ruby, facilitando el hecho de crear CSS para nuestras página. Una de las grandes ventajas que tiene SASS es el hecho de que se puede programar, declarar y asignar variables entre otras cosas. Esto hace que a la hora de cambiar algo sea mucho más sencillo. Para poder usar SASS hay que usar un preprocesador de código como puede ser PrePros.
